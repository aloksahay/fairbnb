# fairbnb
Built at ETHGlobal Cannes 2025

## Project Structure

This project consists of three main components:

- **`ios-client/`** - iOS Swift application (frontend)
- **`api/`** - API server (middleware/gateway)
- **`backend/`** - Backend services and data processing
